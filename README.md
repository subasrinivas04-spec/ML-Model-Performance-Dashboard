# Machine Learning Model Performance Dashboard

## Project Overview
This project presents an interactive Machine Learning Model Performance Dashboard developed using Microsoft Power BI. The dashboard visualizes important evaluation metrics to analyze the performance of a machine learning classification model and monitor prediction results.

## Key Features
- Total Predictions overview
- Model Accuracy and Error Rate indicators
- Prediction Accuracy Distribution
- Errors by Department
- Prediction Bias by Gender
- Confidence Score Distribution
- Accuracy Trend by Month-Year
- Confusion Matrix for model evaluation
- Interactive filters for Department, Gender, and Date
- Drill-through functionality for detailed prediction records

## Tools and Technologies
- Microsoft Power BI
- Data Visualization Techniques
- Machine Learning Evaluation Metrics

## Project Purpose
The purpose of this dashboard is to provide a clear and interactive way to monitor machine learning model performance. It helps identify prediction errors, detect bias in predictions, and support data-driven improvements in predictive systems.
